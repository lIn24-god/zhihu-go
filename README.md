# 蓝山GO组寒假考核项目一：知乎社区

## 实现功能

### 基础功能

1. 基本的用户系统：用户的注册与登录，用户的基本信息，基本的用户鉴权，密码的加密，用户个人信息的更改
2. 发布文章与评论：发布/删除/更新/获取文章，对文章进行评论，对文章进行点赞
3. 关注功能：关注与取消关注，关注列表与关注者列表
4. 内容的搜索：利用MySQL的全文索引功能实现搜索功能
5. ⽂章状态(草稿，已删除，未删除)
6. 管理员禁言
7. 防刷机制(点赞频率，评论频率)
